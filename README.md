### [RTQ描述](https://github.com/OS-Q/RTQ/wiki) 

Q系统包括大量的嵌入式边缘设备，这些设备的开发工具和应用层代码不尽相同，需要通过底层系统来进一步归一化封装，实现系统语言的统一和逻辑功能复用。
